core_logic — یادداشت کوتاه

What this folder holds:
این پوشه چه چیزی دارد:
- Small rules that decide if a thesis can move from one status to another.
- قوانین کوچک که تعیین می‌کنند آیا می‌شود وضعیت پایان‌نامه را تغییر داد یا نه.

Files:
فایل‌ها:
- thesis_status.py — defines Draft and Submitted and a checker function.
- thesis_status.py — وضعیت‌های Draft و Submitted و یک تابع بررسی را تعریف می‌کند.
